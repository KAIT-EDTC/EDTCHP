# 学祭 予約状況確認（静的サイトサンプル）

概要: JSONファイルから予約枠を読み込み、空き状況を表示するシンプルな静的サイトのサンプルです。

## ファイル構成

- `index.html` - メインの静的ページ
- `style.css` - スタイル
- `script.js` - JSONを読み込んでDOMに反映する処理、JSON編集機能
- `reservations.json` - 予約データ（静的）
- `save-reservations.php` - JSON保存用PHPスクリプト

## 機能

### ユーザー向け表示
- 予約枠の空き状況を表形式で表示
- 空き状況に応じた色分け（空きあり/残りわずか/満席）

### 管理者向け機能
- 予約一覧の表示（満席の表示/非表示切り替え可能）
- **JSON編集モード**: 管理者ページから直接予約データを編集可能
  - 時間枠の追加・編集・削除
  - 定員と予約済み人数の変更
  - リアルタイムでJSONプレビューを表示
  - 保存前のバックアップ自動作成

## 使い方

### 閲覧のみ（静的サイト）

1. ローカルで確認するには、`index.html` をブラウザで開きます。ただし、ブラウザのCORS制約でローカルファイルfetchが制限される場合があります。簡易的に確認するにはローカルサーバーを使ってください。

   例（PowerShell）:

   ```powershell
   # Pythonがある場合
   python -m http.server 8000
   # または (PowerShell) .NET Core がある場合
   dotnet tool install --global dotnet-serve; dotnet-serve -p 8000
   ```

2. GitHub Pages に置く場合、リポジトリにこのファイル群をコミットして Pages を有効化すればOKです。

### JSON編集機能を使う場合

JSON編集機能を使うにはPHPが動作するサーバーが必要です。

1. PHPが実行できるサーバー環境を用意
2. すべてのファイルをサーバーにアップロード
3. ブラウザで`index.html`にアクセス
4. 管理者向けセクションの「JSON編集モード」ボタンをクリック
5. 予約枠を編集して「保存」ボタンをクリック

**セキュリティに関する注意:**
- 本番環境では`save-reservations.php`に認証機能を追加することを強く推奨します
- 現在の実装では誰でも編集可能なため、管理者のみアクセスできるよう制限してください

## カスタマイズ

- `reservations.json` を直接編集して時間帯や定員を変更（または管理画面から編集）
- `script.js` のステータス判定や表示文言を変更
- `style.css` で見た目をカスタマイズ

## バックアップ

JSONを保存すると、自動的に `reservations_backup_YYYYMMDDHHMMSS.json` という形式でバックアップファイルが作成されます。

## 注意

- 個人情報は扱わない設計です。Web上では閲覧のみ想定しています。
- JSON編集機能は管理者のみが使用できるよう、適切なアクセス制御を設定してください。
